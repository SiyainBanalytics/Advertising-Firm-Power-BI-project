# Advertising-Firm-Power-BI-project
Built interactive dashboard for Adventure Works manufacturing company to track KPIs including sales, revenue, 
profit, and returns across multiple regions 
Connected and transformed raw CSV data using Power Query, creating a relational data model with multiple 
datasets (transactions, products, customers, territories)  
Developed DAX measures and calculated columns for advanced analytics including regional performance 
comparison, product-level trends, and high-value customer identification
